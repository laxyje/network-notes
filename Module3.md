# Module 3 - Wireless and Mobile Networks

## What I learned

### Wireless networks
Mobile phones use radio waves to communicate with towers and antennas.  
This allows people to make calls, send messages, and use mobile internet.

### Cellular networks
Cellular networks are the mobile networks used by phones.  
Common terms like 3G, 4G, 4G-LTE, and 5G describe different generations of mobile network technology.  
These networks are designed for faster voice and data communication.

### Wi-Fi
Wi-Fi allows a phone or other device to connect to a local wireless network and the internet.  
A device must be within range of a wireless access point or hotspot to use Wi-Fi.

### Bluetooth
Bluetooth is a short-range wireless technology used to connect nearby devices.  
It is often used for headphones, speakers, keyboards, smartwatches, and car systems.

### NFC
NFC stands for Near Field Communication.  
It allows devices to exchange data when they are very close to each other, usually only a few centimeters apart.

### Mobile device connectivity
Most mobile devices can connect to Wi-Fi networks.  
When Wi-Fi is available, the device may use it instead of mobile data.

### Wi-Fi security
It is important to use secure Wi-Fi connections.  
Good practices include:
- not sending passwords in plain text
- using a VPN when possible
- securing home networks
- using WPA2 or stronger encryption

### Mobile operating systems
Two common mobile operating systems are Android and Apple iOS.  
They allow users to connect to Wi-Fi and manage wireless settings.

### Bluetooth pairing
Bluetooth pairing happens when two devices connect and recognize each other so they can share data or services.  
Sometimes a PIN is required to complete the pairing securely.

### Discoverable mode
A Bluetooth device in discoverable mode can be found by other devices nearby.  
It shares basic information like its name, class, and supported services.

## Key terms

- **Wireless network** – a network that uses radio waves instead of cables  
- **Cellular network** – a mobile phone network used for calls, texts, and data  
- **3G / 4G / 5G** – different generations of mobile network technology  
- **Wi-Fi** – wireless access to a local network and the internet  
- **Hotspot** – a place or device that provides Wi-Fi access  
- **Bluetooth** – short-range wireless connection between nearby devices  
- **NFC** – very short-range wireless communication, usually within a few centimeters  
- **VPN** – a secure connection used to protect data over a network  
- **WPA2** – a Wi-Fi security standard used to protect wireless access  
- **Pairing** – the process of connecting Bluetooth devices together  
- **Discoverable mode** – a Bluetooth setting that allows other devices to find it
