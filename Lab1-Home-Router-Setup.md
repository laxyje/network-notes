# Lab 1 - Home Router Setup

## Goal
Set up a small home network in Packet Tracer.

## What I did
- Connected the cable modem, router, PCs, TV, and laptop
- Connected the router to the modem through the Internet/WAN port
- Connected wired PCs to LAN ports
- Used DHCP on the PCs to get IP addresses automatically
- Logged into the router GUI
- Changed the default admin password
- Set the maximum number of DHCP users to 10
- Configured a wireless network with the SSID `MyHome`
- Enabled WPA2 Personal security
- Set a wireless passphrase
- Connected the laptop to the Wi-Fi network
- Verified internet connectivity from the laptop and both PCs

## What I learned
- A home router connects local devices to the internet
- LAN ports are for local wired devices
- The Internet/WAN port connects the router to the modem
- DHCP can automatically assign IP addresses
- Default router passwords should be changed
- WPA2 helps secure a wireless network
- A client needs the correct SSID and password to join Wi-Fi
- Connectivity can be tested by loading a webpage

## Tools
- Cisco Packet Tracer

## Result
The home network was successfully configured and all devices were able to access the internet.
