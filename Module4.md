# Module 4 - Build a Home Network

## What I learned

### Home network basics
A home network usually connects many devices through one router.  
The router connects the home network to the internet and allows devices to communicate with each other.

### LAN and Internet/WAN
A home router usually has two main types of ports:
- **LAN / Ethernet ports** – used for local devices like PCs, consoles, printers, and TVs
- **Internet / WAN port** – used to connect the router to the modem or internet provider

### Home wireless network
A home WLAN allows devices such as phones, tablets, laptops, smart TVs, and consoles to connect wirelessly.  
This makes the network more flexible and convenient for everyday use.

### Network technologies in the home
Home networks can use both wireless and wired technologies.
- **Wireless** includes Wi-Fi and Bluetooth
- **Wired** includes Ethernet cables such as Cat 5e

### Wireless frequencies
Wi-Fi commonly uses the **2.4 GHz** and **5 GHz** frequency bands.  
Bluetooth usually uses the **2.4 GHz** band and is meant for short-range communication.

### Wi-Fi standards
Wi-Fi is based on the **IEEE 802.11** standards.  
Different standards support different speeds and compatibility options.

### Important Wi-Fi settings
Some important wireless router settings include:
- **Network mode** – decides which Wi-Fi standards are supported
- **SSID** – the name of the wireless network
- **Standard channel** – the channel used for wireless communication
- **SSID broadcast** – decides if the Wi-Fi name is visible to nearby devices

### Router setup
When setting up a router for the first time, a computer is usually connected to a **LAN port**, not to the Internet/WAN port.  
The router often assigns IP addresses automatically using **DHCP**.

### Security and design choices
When configuring a home network, it is important to think about:
- what the network name should be
- what devices need to connect
- whether older devices need mixed or legacy mode
- whether guest access should be allowed

It is not a good idea to use the router model or brand in the SSID.  
Hiding the SSID does not provide real security by itself.

### MAC filtering and guest access
Some routers allow **MAC filtering**, which can limit access to specific devices.  
Routers may also offer a **guest network**, which allows internet access without giving full access to the home network.

### Hands-on practice
In this module, I also set up a small home network in the program/lab environment.  
This helped me understand how a router, local devices, Wi-Fi settings, and internet connectivity work together in practice.

## Key terms

- **Home network** – a local network used in a house or apartment  
- **Router** – a device that connects the home network to the internet  
- **LAN** – the local part of the network for devices inside the home  
- **WAN / Internet port** – the router port used to connect to the outside network  
- **Ethernet port** – a port for wired local devices  
- **SSID** – the name of a Wi-Fi network  
- **DHCP** – a service that automatically gives IP addresses to devices  
- **802.11** – the wireless standard family used by Wi-Fi  
- **2.4 GHz / 5 GHz** – common Wi-Fi frequency bands  
- **MAC filtering** – allowing or blocking devices based on their MAC address  
- **Guest network** – a separate network for visitors with limited access  
- **Mixed mode / legacy mode** – settings that help older devices connect
